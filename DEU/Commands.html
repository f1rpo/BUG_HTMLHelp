Tastaturkürzel der BUG Mod

Nachfolgend finden Sie die Tastenkombinationen, die Sie benötigen werden, um die BUG Mod effektiv zu nutzen. Es gibt nicht viele, so dass sie relativ einfach zu merken sein sollten.

Please refer to the "Shortcuts" category of Sevopedia for a list that also includes BULL and Taurus commands.

  • Strg + Alt + O - Öffnet den Optionsbildschirm der BUG Mod.

  • Strg + Alt + F1 - Öffnet die Hilfe-Datei der BUG Mod.

  • Strg + Alt + R / Alt + M - Eine Erinnerung schreiben.

  • Strg + Alt + N - Debug-Funktion zum Testen von Benennungsregeln für Einheiten.

  • Alt + L - AutoLog starten/stoppen.

  • Alt + B - Kampfstatistik ins Protokoll schreiben.

  • Alt + E - Einen eigenen Eintrag ins Protokoll schreiben.
