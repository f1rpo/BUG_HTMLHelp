Support for Networked Multiplayer

Networked multiplayer modes require the game to run perfectly synchronized on the host and all guests. Two BULL options are known to cause games to go out of sync unless either everyone keeps them enabled or everyone keeps them disabled:

  1. Pre-Chop
    • Forests
    • Improvements
  2. Sentry Healing
    • Enabled
    • Only in Neutral Territory
Note that some rare bugs affecting synchronization exist already in BtS and are not (so far) fixed by Taurus. That is, the occasional non-reproducible out-of-sync error does not imply that BULL is causing problems.
