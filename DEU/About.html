About this Help

Notes by f1rpo

This help file has not been updated since BUG 3.5 (Oct 2008), whereas Advanced Civ and K-Mod are based on BUG 4.5, include many features from BULL (which didn't exist yet in 2008) and have added their own options to the BUG menu. The best documentation for all those options is, arguably, the context help shown on the menu. (Much of the layout and text has been revised by AdvCiv.) For an list of additions after BUG 3.5, you might consult AdvCiv\BUG Doc\BUG Changelog.txt and the start of the K-Mod chapter in the AdvCiv manual.

I'm inserting below the contents of the Credits tab of BUG 4.5. K-Mod removed this tab from the BUG menu, and I agree that's not the best place for it, but I do want to include up-to-date credits somewhere. It's probably not quite complete and includes some BULL features that AdvCiv never adopted, e.g. the MapFinder.
