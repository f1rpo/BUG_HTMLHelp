Included Mods

The BUG Mod would no exist if not for the hard work already started by the rest of the Civilization Modding Community. We have included within this help file the original ReadMe files (modified to conform to a unique format) of the Mods we have included in the BUG Mod. If we have change the original Mod in anyway, we have tried to document it, but sometimes things can slip through the cracks! If you feel more information is needed, we left something or someone out, or if you have any other suggestions, then please contact us.
