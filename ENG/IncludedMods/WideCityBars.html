Wide City Bars (WCB)

by asioasioasio


This modification will extend the City Bar by around 25% it's normal size. This will allow for longer names many users experience when using translations of Civ4.

(Deleted the instructions for removing this mod component because Taurus allows it to be toggled via the Map tab of the mod options screen.)
