Wide City Bars (WCB)

by asioasioasio


This modification will extend the City Bar by around 25% it's normal size. This will allow for longer names many users experience when using translations of Civ4.


Instructions

This mod can't be disabled in the BUG Options Screen (Alt+Ctrl+O).
If you would like to remove this mod, simply rename (or delete) the 'Art\Interface\Citybar' folder, located in the '..\Documents\My Games\Beyond the Sword\CustomAssets' folder ('..\Documents\My Games\Beyond the Sword\MOD\BUG Mod\Assets' folder if you have installed the BUG Mod as a mod).
